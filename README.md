# mevn

> A MEVN Stack Sample

## Build Setup

``` bash
# Installation
$ npm install # or yarn install

# Run backend server
$ cd server
$ npm run dev

# Run frontend client
$ cd client
$ npm run dev
```

## Application

Server (Express App) will run on `http://localhost:3001`
Client (Nuxt.js App) will run on `http://localhost:3000`
